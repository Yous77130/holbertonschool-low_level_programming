# Simple Calculator

Ce projet est une calculatrice interactive basée sur le terminal, écrite en langage C.

## Compilation
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator

## Utilisation
1. Lancez : ./calculator
2. Choisissez une opération (1-4) ou 0 pour quitter.
3. Entrez les nombres.

## Limitations
Ne gère pas les entrées non-numériques.
